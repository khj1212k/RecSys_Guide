# 01. Traditional Models

This section covers the foundational algorithms in Recommender Systems.

## Categories

### 1. [Collaborative Filtering](./01_Collaborative_Filtering/README.md)

Leverages user-item interaction data (ratings, clicks) to find patterns.

- **Memory-based**: Uses the raw data directly (Nearest Neighbors).
- **Model-based**: Learns a predictive model from the data (Matrix Factorization).

### 2. [Content-based Filtering](./02_Content_Based_Filtering/README.md)

Recommends items similar to those a user liked in the past, based on item attributes (metadata, text description).
