# MLP-based Models

Multi-Layer Perceptron (MLP) is the most basic form of Deep Learning. In recommender systems, it is primarily used to replace the linear "Dot Product" interaction calculation used in Matrix Factorization with "non-linear learning via neural networks."

## Types

### 1. [NCF (Neural Collaborative Filtering)](./01_NCF.md)

- Generalizes Matrix Factorization. Concatenates user and item vectors and passes them through an MLP to learn complex non-linear interactions.

### 2. [Wide & Deep Learning](./02_Wide_and_Deep.md)

- Proposed by Google. Combines the memorization capability of linear models (Wide) with the generalization capability of deep models (Deep).
