# LLM-based Models

Large Language Models (LLMs) bring external knowledge and zero-shot capabilities to Recommender Systems.

## Models

### 1. [LLM4Rec](./01_LLM4Rec.md)

General paradigm of using LLMs (like GPT-4, LLaMA) as a recommender or an enhancer for existing recommenders.

### 2. [P5 (Pretrain, Personalized, Prompt, Predict, Recommendation)](./02_P5.md)

A unified text-to-text framework where all recommendation tasks (Rating, Explanation, review) are formulated as prompts.
